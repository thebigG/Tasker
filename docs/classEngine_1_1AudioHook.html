void AudioHook::update ( )
overridevirtualslot

AudioHook::update updates the state of AudioHook to productive if the audio volume(level) is above audioThreshold, otherwise it sets the state to unproductive. It also profiles the audio device's if it hasn't been profiled yet.

Note
Note that this function is called every time the AudioDevice::audioRead signal is sent.
