Detailed Description

The Engine::XListener class This class has the capiblity of detecting mouse and keyboard interaction.

Note
Note that the detecion of mouse and keyboard input(globally throughout the system) is made possible by the amazing libuiiohook library. You can check out the library here https://github.com/kwhat/libuiohook
