int KeyboardListener::startListening ( unsigned long int  delay = 30)

KeyboardListener::startListening holds the core logic of this listener. This is a routine on an infinite loop that waits for keyboard input to come in. Once it receives(detects) keyboard input, it will hold an active state for the listener for the number of sconds that is delay.

Parameters
delayFor how long do we keep the active state. The default value is 30 seconds.
