Detailed Description

The Engine::Hook class is the base class for all of the hooks on Tasker. It provides a stateful(with productive and unproductive states) interface that any child class may extend and build upon.

Note
If, for whatever reason, we decide to add HookType values make sure to add them BEFORE none. The none type is used for bounds checking.
