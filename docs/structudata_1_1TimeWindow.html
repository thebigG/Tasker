Detailed Description

The TimeWindow struct represents a span of time(such as a week) that the user has a number of tasks(defined by frequency) to complete for this partifuclar TimeWindow, whether that'd be a week or a month. startDate is the beginning of the TimeWindow, while endDate is the end of that TimeWindow(the seventh day in the case of a week.)

bool udata::TimeWindow::isDone ( )
inline

isDone represents whether or not this TimeWindow's sessions have all been completed or not. The number of sessions that MUST be compeleted for TimeWindow is determined by the frequency of this TimeWindow. This is particularly useful for the UI when it wants to know whether it can add sessions or not to a particular TimeWindow.

Returns
true if all sessions have been completed. false otherwise.
